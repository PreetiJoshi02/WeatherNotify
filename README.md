## 🌦 WeatherNotify App

A simple and responsive weather notification web application built using **Node.js**, **HTML**, **CSS**, and **JavaScript**.  
It allows users to check real-time weather information by entering a city name.

---
## 📌 Project Overview

WeatherNotify is a simple and user-friendly web application that provides real-time weather updates for any city.  
Users can enter a city name and instantly get current weather details such as temperature, conditions, and more.

The application is built using a **Node.js backend** and a **responsive frontend** with HTML, CSS, and JavaScript.  
It fetches live weather data from an external API and displays it in a clean and visually appealing interface.


## 🚀 Features

- 🌍 Get weather by city name  
- ⚡ Fast and lightweight  
- 🎨 Clean UI with gradient design  
- 🔔 Weather notification display  
- 🔐 Secure API key handling using environment variables  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js  
- **API:** Weather API

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/PreetiJoshi02/WeatherNotify.git

cd WeatherNotify
```

---

### 2. Install dependencies
```bash
npm install
```

### 3. Create a `.env` file
In the root folder, create a file named `.env` and add:

```bash
API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
node server.js
```

Now open your browser and go to:

```bash
http://localhost:3000
```
